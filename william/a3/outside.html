Common Tags Outside of <body>

<body> Tags Definition Examples
<!DOCTYPE html> Designates the code as HTML. N/A
<html lang="en"> Designates the language as English If the browser has any translation features, the language can help what to translate the page from.
<meta charset="utf-8"> Designates the character set as the standard Unicode. N/A
<title>...</title> Sets the title of the webpage.
<style>...</style> Section that describes the internal CSS. <style> td#example { background-color: darkmagenta; }</style>
<link rel="stylesheet" href="style.css"> Section that gets styles from an external file. The homepage and this page point to a common style.css file that controls the look of these pages.
<link rel="icon" type="image/x-icon" href="favicon.ico"> Designates the favicon of the webpage.
