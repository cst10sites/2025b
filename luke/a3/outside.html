Tag Definition Examples:
<html lang=" "> tells what language to translate to This page is set for "en" for English. This tag helps websites such as Google Translate know the default for this page is English.
<!DOCTYPE html> tells that html coding is being used everything here was made with html
<head> The head of the page which stores many components (e.g. title, link, style, etc.) <head>
<meta charset="utf-8">
<title>Assignment 3</title>
<link rel="stylesheet" href="style.css">
<link rel="icon" type="image/x-icon" href="favicon.ico">
</head>
<meta charset="UTF-8"> Specifies the character encoding for the page N/A
<title> Displays a title for the tab header
<style> Section that indicates internal CSS <style>
body {background-color: black;}
</style>
<link rel="stylesheet" href="style.css"> Tag that indicates external CSS file The homepage and this page point to a common "style.css" file that controls the look of these pages
<link rel="icon" type="image/x-icon" href="favicon.ico"> Sets an image (favicon) beside the title N/A
