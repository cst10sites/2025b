HTML Guide

Common HTML Tags Outside <body>

Tag Definition Examples
<!DOCTYPE html> Ensures the webrowser of the type of HTML being used This website is an example of using <!DOCTYPE html>
<html lang="en"> Determines the language of the page E.G. English is the language of this website
<meta charset="UTF-8"> Used to specify the character encoding for the HTML document. Defines the character encoding
<title></title> Used to specify the title of a webpage
<style></style> Section that determines internal CSS This sentence uses internal CSS modifications
<link rel="stylesheet" href="style.css"> Tag that indicates external CSS file The homepage and this page point to a common "style.css" file that controls the look of these pages
<link rel="icon" type="image/x-icon" href="favicon.ico"> Used to add a favicon to the webpage
<meta http-equiv="refresh" content="0; url=$URL"> Is used to instruct a web browser to automatically refresh the current web page or frame after a specified time interval. No example applicable
