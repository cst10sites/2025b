Common HTML tags outside <body>

<head> Tags Definition Examples (if applicable)
<!DOCTYPE html> Used to define what type of document it is for the browser. No example applicable
<html lang="en-US">...</html> Defines the language of the page, and is where the HTML is located This page is set to "en-US", so if your browser's default language is English, the translate function will not activate.
<meta charset="UTF-8"> Defines the character library (UTF-8 is the best one for English/Latin languages) No example applicable
<title> Defines the title of the page.
<style> Used to add CSS to a website without an external CSS file. <style> .black { background-color: black; color: white; } </style>
<link rel="stylesheet" href="styles.css"> Used to add an external CSS file to the page. Both the homepage and this page are connected to the CSS file
<link rel="icon" type="image/x-icon" href="/images/favicon.ico"> Used to add a favicon (or the icon on the tab) to your webpage.
