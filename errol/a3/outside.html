Common HTML Tags Outside <body>

Tags Definition Examples
<!DOCTYPE HTML> It is information to the browser about what document type to expect. No examples.
<html lang="en"> Specifies the language of the element's content. This page is set to "en" for English. This tag helps websites such as Google Translate know the default language for this page is English.
<meta charset="UTF-8"> Allows you to use special characters like emoji and etc. No examples.
<title></title> Defines the title of the document.
<style></style> Section that describes the internal CSS of the page. This background is green. It is controlled by a class that is defined in style.
<link rel="stylesheet" src="file"> Tag that adds a favicon.
<link rel="stylesheet" src="link"> Link a file to another like linking CSS. It is not visible
