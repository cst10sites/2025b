Common HTML tags outside <body>

Common HTML tags outside <body> Common HTML tags inside <body>

<body> Tags Defenition Examples
<!DOCTYPE html> Defines document type and version No examples
<meta charset="UTF-8"> Allows you to use other characters not on the traditional character set on your webpage No examples
<html lang="en"> Specifies the language This page is set to “en” for English. This tag helps websites such as Google Translate know the default language for this page is English.
<title>...</title> Gives your webpage a title
<link rel="stylesheet" href="style.css"> Tag that points to an external CSS file The homepage and this page point to a common style.css file that controls the look of these pages.
<link rel="icon" type="image/x-icon" href="name"> Tag that adds a favicon
<style>...</style> Section that describes the internal CSS of the page    The style is making the background black and the font white
<meta http-equiv="refresh" content="0; url=$URL"> Redirects webpage after a period of time No examples
