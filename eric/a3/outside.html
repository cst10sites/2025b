Tag Definition Examples
<table> Creates a table <table></table>
<tr>, <td>, <th> Table row, data cell, header cell <tr><td>Data</td></tr>
<form> Form container for user input <form action="submit.php"></form>
<input> Input field <input type="text">
<button> Clickable button <button>Click Me</button>
<label> Label for form elements <label for="name">Name:</label>
<section> Defines a section in a document <section>Content</section>
<article> Self-contained composition <article>Post</article>
<footer> Footer section <footer>© 2025</footer>
<nav> Defines navigation links <nav><a href="#">Home</a></nav>
