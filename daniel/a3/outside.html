Tag Definition Examples
<!DOCTYPE html> Defines HTML5 document type <!DOCTYPE html>
<html lang="en"> Root element (language set to English) <html lang="en">
<meta charset="UTF-8"> Sets character encoding (supports all symbols) <meta charset="UTF-8">
<title> Page title (shown in browser tab) <title>My Website</title>
<style> Internal CSS styling <style> body {background: #f0f0f0;} </style>
<link rel="stylesheet"> Links to an external CSS file <link rel="stylesheet" href="styles.css">
<link rel="icon"> Sets website favicon (tiny icon in tab) <link rel="icon" href="favicon.ico">
