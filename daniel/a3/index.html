🔍 Common HTML Tags Inside <body>

🚀 Learn About Head Tags →
Tag Definition Examples
<h1> to <h6> Headings (h1 = most important, h6 = least) <h1>Main Title</h1>
<h2>Subheading</h2>
<p> Paragraph for text <p>Hello world!</p>
<a> Hyperlink to another page <a href="page.html">Click me!</a>
<ul> & <li> Unordered (bullet) list <ul><li>Apple</li><li>Banana</li></ul>
<ol> & <li> Ordered (numbered) list <ol><li>First</li><li>Second</li></ol>
<img> Embeds an image <img src="cat.jpg" alt="Cute cat">
<div> Container for grouping content <div>Content here</div>
<span> Inline container for styling <span style="color:red">Red text</span>
<table>, <tr>, <td>, <th> Creates a table structure <table><tr><th>Name</th></tr><tr><td>John</td></tr></table>
<strong> & <em> Bold and italic emphasis <strong>Bold</strong> & <em>Italic</em>
