HTML Guide

Common HTML Tags outside <body>

Tag Definition Example
<!DOCTYPE html> Informs the browser what language is being used, in this case HTML No example
<html lang"en"> Sets the langauge of the document to English The default language of this page is English
<meta charset="UTF-8"> Tells browser what character encoding to use for the document No example
<title>...</title> Sets the title for the page
<style> </style> Section that describes the internal CSS of the page <style>
.color {
color: sandybrown;
background-color: forestgreen;
}
</style>

The code above makes this cell green with a brown font.
<link rel="stylesheet" href="name-of-your-file.css"> Inserts CSS from a seperate document This page as well as the homepage share the same CSS from the same file
<link rel="icon" type="image/x-icon" href="your-file.ico"> Tag that adds a favicon, the image next the title
