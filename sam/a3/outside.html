HTML/CSS Guide

Common HTML tags inside <head>

Tag Definition Example
<!DOCTYPE html> Tells the browser that the website is made in HTML. No examples.
<html lang=en> Tells the browser that the website is in English. This website is set to English.
<meta charset="UTF-8"> Tells the browser that the website is using the UTF-8 encoding standard. No examples.
<title> Defines the websites title.
<style> Lets one add internal CSS. This cell is styled using internal CSS.
<link rel="stylesheet" href="style.css"> Lets you add external CSS. This website for the most part uses external CSS.
<link rel="icon" href="favicon.ico" type="image/x-icon"> Lets you add a favicon. No example.
