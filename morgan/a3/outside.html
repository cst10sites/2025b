Inside HTML

Common HTML Tag outside body

<head>Tags Definitions Example(s) if applicable
< !DOCTYPE html > Shows the web browser what version of HTML the page is written in. It helps the browser understand how to interpret the document. No examples*
<html lang="en"> This is used to specify what language the content in the website is in. This page is set to “en” for English. This tag helps websites such as Google Translate know the default language for this page is English
<meta charset="UTF-8"> This tag specifies the character encoding for the document making sure it gets interpreted correctly and displayed how it was entended to. No examples*
<meta http-equiv="refresh" content="0; url=$URL"> This tag redirects the webpage after a certain amount of time specified. No examples*
<title>...</title> This is used to write the title of the website that appears in the tab of the website (The writing that says "Assignement 3")
<style>...</style> Section that describes the internal CSS of the page Show <style>...</style> that makes this cell background black using id or group
<link rel="stylesheet" href="styles.css"> Tag that points to an external CSS file. The homepage and this page point to a common style.css file that controls the look of these pages.
<link rel="icon" type="image/x-icon" href="...(link to favicon file)"> Tag that adds a favicon (The robot picture next to the writing)
