  3. Number three

<img src="..." alt="name of image"> Adds your chosen image to the website! Put the link into src and the name you want to call the image into alt.

<h6>...</h6> 		Creates a header with a size corresponding to the natural number beside h, from 1 up to 6. 1 is the biggest size 6 is the smallest.

<a href="...">...</a> Creates a hyperlink of the word(s) in between the anchor tag. The href is the link you will be redirected to if you click the hyperlink (add target="_blank" to open the link in a new tab/window.) Click here to be redirected to google.
<p>...</p> Allows yout to write text in between the p tags that will appear on the website.

This is a paragraph. you can change the size and font using CSS!
<iframe name="..." src="..."></iframe> Allows you to embed another webpage or element onto the website. (like a Youtube video.)
<em>...</em> and <strong>...</strong> Adds emphasis (<em>) and strong emphasis (<strong>) to the word(s) in between the tags.     This is emphasis.     This is strong emphasis.
<span>...</span> An inline container used to group and style parts of text. Changes the visual element of one part of text Only this word is green
