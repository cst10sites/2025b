Common HTML Tags outside <body>

<body> Tags Definition Examples (if possible)
<!DOCTYPE html> This tag is an essential part of HTML documents. It is used to specify the HTML version and ensure that the web browser renders the page correctly. No examples*
<html lang="en"> This tag is used to declare the language of the content within an element. When you set lang="en" in the html tag, it specifies that the content of the document is in English. This page is set to english ("en"). This tag helps websites know the default language for this page is English.
<meta charset="UTF-8"> Tells browser what character encoding to use for the document. No examples*
<title>...</title> This tag makes the title of your webpage.
<style>...</style> Section that describes the internal CSS of the page I used id and the style CSS to change the background colour of this box
<link rel="stylesheet" href="styles.css"> Tag that points to an external CSS file The homepage and this page point to a common style.css file that controls the look of these pages.
<link rel="icon" type="image/x-icon" href="your .icofile"> Tag that adds a favicon
