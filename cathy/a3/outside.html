<body> Tags Definition Example(s)
<!DOCTYPE html> all HTML documents must start with this tag no examples*
<link rel="stylesheet" href="styles.css"> points to an external CSS file this homepage and this page point to a common style.css file that controls the look of these pages
<html lang="en"> sets to "en" for English, helps websites know the default language for this page is English no examples*
<meta charset="UTF-8"> typically used to specify character set, page description, keywords, author of the document, and viewport settings no examples*
<title> required title of the document, must be text-only, and it is shown in the browser's title bar or in the page's tab
<style> section that describes the internal CSS of the page no examples*
<link rel="icon" type="image/x-icon" href="dizzy.ico"> adds a favicon
