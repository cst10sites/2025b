<h1> Heading These tags are used to resize text, whether it's to draw attention to the content or to signal the start of a new section in the website. There are variants of the heading tag, called <h1> through <h6>. The smaller the number, the larger the text!

<em> Emphasis This tag is used for one thing. Emphasis. It can make text bold, and screen reading programs will treat the text as such. This is normal text compared to Emphasised text.
<table> Table This tag makes an array of cells that can all be filled in. use <tr> inside the larger tag for table rows, and within that tag, you can add <th> for headings and <td> for individual cells
