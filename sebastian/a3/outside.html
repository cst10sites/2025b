<title>...</title> Sets the title of the website

Using IDs and <div> elements, we can make the background of this square black.
<link rel="stylesheet" type="text/css" href="filename.css"> Tag that points to an external CSS file The homepage and this page point to a common style.css file that controls the look of these pages.
<link rel="icon" type="image/x-icon" href="filename.ico"> Tag that adds a favicon
