Head Tag Definition Example
1. <!DOCTYPE html> 
										<!DOCTYPE html>
...
Declares that this file is an html file. Generally useless and most browsers will treat a .html file as an html file no matter what. Even if you put it at the start of, say, a .txt file it'll still be read in plain text even though you have that at the start. N/A
2. <meta> 
										<meta charset="utf-8">
Does a bunch of things including descriptions for search engines but the only thing I'm allowed to know is that you can use it to set the character set. It's also useful for mobile support but they don't want you to know that. N/A
3. <html> 
										<html>
	...
</html>
State that this stuff is HTML. Completely useless save for the lang="..." box that lets your browser know whether or not to engage the translator program thingy N/A
4. <style>

	...
</style>
box for CSS. N/A but in green

										<link rel="stylesheet" href="style.css">
