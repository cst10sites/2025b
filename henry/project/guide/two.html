In our first lesson together, we learnt HTML. That was the programming language we were writing that dictates where things go and what the content is. Now we're going to learn CSS, the language that controls how the website looks.

<style>...</style>

This is the style tag. Inside of it we write our CSS. Isn't that neat? A whole programming language inside of a programming language. Here: I'll write you some so you can get a feel for the structure of it, then we'll break down what it is I just wrote.

Look at that! Our website looks so fancy! But what have I done?

body {
  ...
}

This is the general syntax of CSS. You start with the name of the thing you want to style, and then some curly brackets (These things: {}) And between them, your style code.

  background-color: black;

This is a line of CSS style code. It's written between the curly brackets. Here, we're setting the background colour to be black.
(Side note: It's spelt the American way here, unfortunately we'll have to deal with it as the code doesn't support the alternate spelling.)
Try replacing black with a different colour! (Don't change anything else)

Neat! You've probably noticed the line that just says color. That's the colour of the text. I have no idea why it's named this way, it's very confusing. Anyways, let's try something. I'm going to test you to make sure you've been listening.

I'm going to let you have full control of your code in a second, and you're going to set a <p> tag to have a green background colour. (Don't worry, you can read back through the dialog)

Nice! I don't like this font, though. Let's change it to something else.

  font-family: monospace;

This changes the font to monospace. Be careful with this: It only works with fonts that are preinstalled with most computers. If most people don't have the font, it probably won't work. With that in mind, try a few fonts like "Helvetica" or "Comic Sans MS".

CSS Tips

Example of a CSS statement: 
p {
  background-color: red;
}
REMEMBER TO PUT YOUR CSS IN THE STYLE SECTION: 
<style>
p {
  background-color: red;
}
</style>
