Cheatsheet

Some of the more useful HTML tags.

Body Tag Definition Example
1. <img> 
										<img src="..." style="width: ...; height: ...;" alt="...">
Display an image on your website. Set the image url with src="..." and set the width/height with style="width: 50%; height: 50%;" & alt text (text that shows if you use a screen reader or have image loading disabled)
Unrelated: shout out to fediverse users for making alt text become second-nature in my brain
2. <a> 
											<a href="..." target="...">...</a>
a tag that can link to a url/file and send you there on click. File is set with href="...". Can make open in a new tab by setting the target to _blank. If you match up the target with an iframe's name, it'll open up in that iframe. Frog YouTube
3. <div> & <span> 
											<div class="..." id="..." style="...">...</div>
										
											<span class="..." id="..." style="...">...</span>
doesn't really do anything by default (span especially) div is just a box with display="block" by default & span is just a box with display="inline" by default. It gets interesting when you add css with class, id, & style.
So much better than p tags
4. <ol>, <ul>, & <li> 
											<!-- ol: ordered, ul: unordered -->
<ol>
  <li>...</li>
</ol>
Lists: ol for ordered lists, ul for unordered lists. li is for list items Egg Recipe

  3. Consume raw.

6. <em> 
											<em>...</em>
Emphasis: italic text, emphasis when read with screen reader I feel so important today.
7. <p> 
											<p>...</p>
P tags (paragraph tags) are more of a code readability thing, don't feel bad if you just replace them with divs. Example (I'm lying to you this isn't in a p tag at all but you couldn't tell)
8. <iframe> 
											<iframe src="..." name="..."></frame>
Put a website in a website. Set the website with src="{{url}}". You can force a tag redirects to run in the iframe with the name="{{a tag target}}" (set the a tag's target to the same thing)
9. <table>, <tr>, <th>, & <td> 
											<table>
	<tr>
		<th>Table header in first row.</th>
	</tr>
	<tr>
		<td>Table data point in second row.</td>
	</tr>
</table>
Table. Tr: table row. Th: table header. Td: table data. See: this web page
10. <sup> & <sub> 
											<sup>...</sup> <sub>...</sub>
<sup>: text small and at top like when squaring a number. <sub>: text small and at the bottom like in the formula of a compound. 52, H2O
11. <br> 
											<br>
Breakline, newline, whatever you'd like to call it. Described as \n in some languages, it's the equivalent of hitting enter and typing on another line. Words on one line
