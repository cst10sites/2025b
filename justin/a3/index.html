<ol>, <ul>, <li> These are list tags. <ol> (standing for Ordered List) displays numbers, while <ul> (standing for Unordered List) is displayed as a bullet point form list with no numbers or indications. To add list items, use the <li> tag for each item.

<img src="link" alt="pic"> The <img> tag allows you to add images. To add the image, you must link the image with src="link". You can either link an URL online with the photo, or from a folder on your files. You can customize the size of the image with style="width=:x%".
<iframe src="link"> The <iframe> tag allows you to embed videos and other. Simmilarly to the preceding tag, you can also alter the size of the embedded video, with the width="x" and height="x" code.
