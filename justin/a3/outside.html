HTML/CSS Guide

Common HTML Tags outside <body>

Tag Definition Examples
<!DOCTYPE html> The doctype in HTML is a declaration that tells a browser which version of HTML a document is written in. No examples (None applicable)
<html lang="en"> Specifies that the language of the document is in a certain language. No examples (None applicable)
<title></title> Sets the title of the page, which will be displayed on the tab of the page. Refer to the title of this page.
<style> A section used to define internal CSS (Cascading Style Sheets) directly in the HTML file. This allows you to stylize tags spcifically. Input <style> in <head> (before <body>), then inside the section put #x (x = any name) followed by a curly bracket ({). Next, press enter, and when you are inside the brackets, type "background-color: black;" and "color: white;" to achieve this effect.
<link rel="styleheet" href="x"> This is a tag used in HTML to link an external stylesheet CSS file to an HTML document, which can modify and stylize your website. The homepage and this page point to a common style.css file that controls the look of these pages.
<link rel="icon" type="x" href="y.ico"> Tag that adds a favicon, the icon that displays on the tab of the website. Must insert a file with the .ico extension to work. Refer to the favicons that are both used by this page and the inside page. Below is the favicon used for this page.
