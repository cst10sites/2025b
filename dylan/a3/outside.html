Common HTML tags outside body

Common tags inside <body>
Tag Definition Examples
<!DOCTYPE html> All HTML documents must start with a <!DOCTYPE> declaration. The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect. No examples*
<html lang="en"> The lang attribute specifies the language of the element's content. This page is set to “en” for English. This tag helps websites such as Google Translate know the default language for this page is English.
<meta charset="UTF-8"> The charset attribute specifies the character encoding for the HTML document. No examples*
<meta http-equiv="refresh" content="0; url=$URL"> Redirect visitors to another URL or page. No examples*
<title>...</title> Defines title of document.
<style></style> Section that describes the internal CSS of the page Show <style>...</style> that makes this cell background black using id or group
<link rel="..." href="..."> Tag that points to an external CSS file The homepage and this page point to a common style.css file that controls the look of these pages.
<link rel="icon" type="image/x-icon" href="..."> Tag that adds a favicon
